📊 Interactive Sales Dashboard (Excel)

🚀 Project Overview

This project is an interactive sales dashboard built using Microsoft Excel. It analyzes sales performance across different regions, products, and time periods.

🎯 Key Features

- Monthly Sales Trend Analysis 📈
- Sales by Region 🌍
- Profit by Product 💰
- Interactive Filters using Slicers 🎛️
- KPI Metrics (Total Sales, Profit, Orders)

🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Cleaning Techniques

🧠 Key Insights

- Identified top-performing regions
- Analyzed monthly sales fluctuations
- Compared product profitability

📌 Conclusion

This dashboard helps in understanding business performance and supports data-driven decision making.
